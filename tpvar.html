tipos de variaveis

como toda liguagem de programação as variaveis são de extrema importancia, nalas podemos armasenar dados.

existem alguns tipos de variaveis:
Number que é para numeros, ela poder receber numeros
infinit e NaN
inteiros ou reais para inteiros =int e reais = float.

se declarar um numero entre aspas ele deixa de ser um numero e se torna uma string

string que são caracteres
boolean que so aceita verdadeiro ou falso
object , array é um tipo de objeto
function.

Como descobrir que tipo é a variavel

para descobrir qual tipo de dado é uma variavel o Javascrypt tem um comando que revela o tipo que é o typeof

conversões

converssões de string para numero usa Number.passInt, Number.passFloat ou só Number.
conversões de numero para string usa x.toString ou String(x)
